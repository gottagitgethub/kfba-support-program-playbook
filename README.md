# 외식업 지원사업 실전 설계도구

강의자료(`kfba-ai-economy-lecture`)와 분리된 별도 실습/지원사업 웹앱입니다.

중장년·시니어 사장님도 쉽게 확인할 수 있도록 다음을 포함합니다.

- 중앙 지원사업: 강한 소상공인 성장지원, 로컬크리에이터, 스마트상점, 온라인 판로·물류, 희망리턴패키지, 소공인 클린제조환경
- 충남/시군 지원사업: 사회보험료, 충남 온라인 플랫폼 판로지원, 시군 경영환경개선, 음식점 위생·입식탁자·조리장 개선
- 쉬운 지원정책: 전기요금·에너지, 카드수수료, 배달·택배비, 노란우산, 풍수해·화재보험, 정책자금 기본안내
- 쉬운 신청 5단계: 사진 → 기본서류 → 견적 → 전화 확인 → 방문/온라인 접수
- 가게 조건 기반 맞춤 추천, 증빙 체크리스트, 비용/우선순위 판단, 신청서 초안 생성 프롬프트

## 실행

```bash
npm install
npm run start
```

브라우저에서 `http://localhost:4175` 접속.

## 검수

```bash
npm run check
```

## 발표/실습 조작

- 분류 이동: ← / →
- 분류 바로가기: 1~9
- 전체화면: F 또는 `전체화면` 버튼
- 마지막 분류는 `localStorage`에 저장됩니다.

## 주의

지원사업의 자격요건, 지원한도, 마감일, 제외업종은 매년·지역별로 바뀝니다. 실제 신청 전에는 반드시 해당 연도·시군 원문 공고를 확인해야 합니다.
